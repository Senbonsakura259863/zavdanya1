Довідник по HTML тегам

‹!DOCTYPE› - призначений для вказівки типу поточного документа.

‹html lang=""› - глобальний атрибут, який допомагає визначити мову елемента:
мову, на якій написані не редаговані елементи,
або мову, на якій користувач повинен редагувати редаговані елементи.

‹head› ‹/head› - призначений для зберігання інших елементів, мета яких - допомогти браузеру в роботі з даними.

‹meta› - зберігає інформацію призначену для браузерів і пошукових систем.

‹title› ‹/title› - відображає у заголовку браузера ім'я поточного html-документа.

‹!--...--› - визначає коментар в документі.

‹body› ‹/body› - призначений для зберігання тіла веб-сторінки (його контенту), що відображається у вікні браузера.

‹h1› до ‹/h6› - заголовки від найбільш важливих до найменш важливих

‹p› ‹/p› - текстовий абзац

‹a› ‹/a› - створює посилання на іншу сторіку(Зовнішне посилання)
або на певний елемент цієї сторінки(Внутрішне посилання).
